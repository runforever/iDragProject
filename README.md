### 使用
``` bash
git clone https://github.com/runforever/iDragProject.git
cd iDragProject
pod install
```

Xcode 打开 `iDragProject.xcworkspace/`，使用 `Cmd + b` 快捷键编译项目，替换图片中的 key，然后 do what your wanna do。
